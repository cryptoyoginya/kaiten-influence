# Kaiten · инфлюенс-маркетинг

Платформа-дашборд для управления посевами у блогеров. Уводит процесс из хаоса
Кайтена в прозрачную доску: **один блогер — одна строка**, бэклог и спринт — это
два вида одной таблицы, а не копипаст между листами.

## Экраны

- **Бэклог** (`/backlog`) — все блогеры по нишам, со всеми столбцами (исследование
  + рабочие поля размещения). Поиск, фильтр по нише, флаг шортлиста.
- **Спринт** (`/sprint`) — размещения текущей недели. Канбан по 10 этапам пайплайна
  (от «креатив согласован в Кайтен» до «аналитика снята») + карточки со всеми
  заполняемыми полями + сводка по экономике (бюджет, прогноз охвата, CPV).

## Данные

Источник — выгрузка лонглиста из Excel. Импортёр нормализует 13 листов в чистый
датасет с дедупликацией по telegram-ссылке:

```bash
pnpm import          # data/channels.json + data/sprints.json из Excel
```

- `data/channels.json` — 169 блогеров, ниши, экономика, флаг шортлиста.
- `data/sprints.json` — спринты и размещения с пайплайном.

## Запуск

```bash
pnpm install
pnpm dev             # http://localhost:3007
```
