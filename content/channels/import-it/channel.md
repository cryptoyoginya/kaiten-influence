## ICP-привязка (из [icp-kaiten.md](../../references/icp-kaiten.md)) — «чётко по ICP»

Канал бьёт в конкретные ICP-роли, боли брать дословно:
- **Руководитель управления по импортозамещению** (Производство): Jira/MS Planner не обновляется → уязвимости ИБ; риск штрафов/прекращения поддержки → паралич. Пруф: сертификат «Промтехстандарт», «Сколково выбирает Kaiten», план импортозамещения 100%.
- **Head of Engineering / Руководитель разработки** (Крупная IT): риск отключения → паралич, задержка релизов, срыв контрактов. Метрика — **uptime %**. Кейс — **X5 из Jira** (kaiten.ru/blog/iz-jira-v-kaiten-case-x5).
- **CIO / Директор по ИТ** (Крупный банк): не соответствует регуляторам финсектора, риск утечки → сертификация РФ, on-prem.
- **Замминистра / цифровизация** (Госсектор): госзакупки, on-prem (облако запрещено), ФСТЭК.
