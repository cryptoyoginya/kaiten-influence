# Голос-профиль: «Уставший техдир»

> ⚠️ **Черновик по косвенным признакам.** Заполнить из 5–7 реальных постов
> (чек-лист в channel.md). До этого voice-match в прогонах помечается
> «по черновому профилю».

## Гипотезы (сверить по реальным постам)

- Название канала задаёт регистр: усталый скепсис опытного руководителя,
  ирония без энтузиазма. Восторженная подача — табу.
- Аудитория senior: термины без пояснений «для маленьких».
- Вероятные приёмы: сухая констатация, самоирония про «ещё один инструмент»,
  опыт вместо обещаний.
- Согласованный формат опроса («кто на какой платформе») — интерактив каналу знаком.

## Как использовать

Голос-тест по creative-diversity.md после занесения корпуса. Для сета этого
канала особенно важен И2 (разработчик-скептик) из панели it-dev: аудитория
Сбера и FrontendConf чувствительна к маркетинговому звону.
